5、JDK 元注解

在JDK中除了java.lang包下有Annotation，在java.lang.annotation下也有几个常用的元Annotation。

这类注解都是用于修饰其他Annotation定义。

**
**

@Retention

用于指定被修饰的Annotation被保留多长时间。

Java 文件三个时期：编译、class、运行。

这个注解默认是class，而反射是在运行期执行的，故而获取不到数据。

我们加上这个注解，就正常了。

image.png

@Document

用于指定被该Annotation修饰的Annotation类将被javadoc工具提取成文档。

@Inherited

@Inherited也是用来修饰其他的Annotation的，被修饰过的Annotation将具有继承性。

A类上未加@MyAnnotation注解，而A基础B，B类上加了@MyAnnotation注解。

如果这个注解上加了@Inherited注解，那么A类也将包含@MyAnnotation注解。
